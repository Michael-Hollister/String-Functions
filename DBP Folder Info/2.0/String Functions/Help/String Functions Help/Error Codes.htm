Error Codes

Here is the full list of error codes to tell you what went wrong in the function.

  Error Codes
Code Error Help
-1 Null string Indicates that a empty string was passed in to the function.
-2 Index out of bounds The index was less than 1 or greater than the length of the string.
-3 Invalid path characters The string contains invalid characters. For more information, look at SF Path_Valid command for the list of invalid characters.
-4 Over maximum path The string is over maximum character count for the path. (Max = 260)
-5 Unknown string The string was in a incorrect format.
