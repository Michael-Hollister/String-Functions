SF Path_GetParentName

Get's the path's parent folder name.

  Syntax 
Return String = SF Path_GetParentName( Path String )
  Parameters 
Path String
The path string to check.

  Returns

Returns the name of the folder containing the file.
Error Codes: -1 (Null string), -3 (Invalid path characters), -4 (Over maximum path), -5 (Unknown string)

  Description

Get's the path's parent folder name. (This does not return the full path)

  Example Code 
String$ = "C:\Documents\Textfile.txt"
Name$ = SF Path_GetParentName(String$)

Print "String evaluated = " + String$
Print "Folder name is " + Name$
Wait Key
