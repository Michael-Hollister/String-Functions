SF Path_ChangeFileExtension

Changes the path's file extension.

  Syntax 
Return String = SF Path_ChangeFileExtension( Path String, Extension String )
  Parameters 
Path String
The path string to check. 
Extension String
The extension to replace the current one in the path.

  Returns

Returns the path string with the new extension.
Error Codes: -1 (Null string), -3 (Invalid path characters), -4 (Over maximum path), -5 (Unknown string)

  Description

Changes the path's file extension. You could optionally put the "." in the extension string if you wish, either way will work.
Also if the extension string is null, then the extension is removed.

  Example Code 
String$ = "C:\Documents\Textfile.txt"
String2$ = "DBPRO\Media\Image.dds"
String3$ = "C:\Documents\Information.txt"

Name$ = SF Path_ChangeFileExtension(String$,"doc")
Name2$ = SF Path_ChangeFileExtension(String2$,".bmp")
Name3$ = SF Path_ChangeFileExtension(String3$,"")

Print "String evaluated = " + String$ + " : New path = " + Name$
Print "String evaluated = " + String2$ + " : New path = " + Name2$
Print "String evaluated = " + String3$ + " : New path = " + Name3$
