SF BoolToStr

This converts the Boolean value to a string representation of the value.

  Syntax 
Return String = SF BoolToStr( Boolean )
  Parameters 
Boolean
The value to convert.

  Returns

Returns a string representation of the value. (True-False).
Error Codes: -5 (Unknown string)

  Description

This converts the Boolean value to a string representation of the value.

  Example Code 
Print SF BoolToStr(0)
Print SF BoolToStr(1)
Print SF BoolToStr(223)
