SF RegEx_GetMatchPosition

Returns the index in the string where the match was successful.

  Syntax 
Return Value As Integer = SF RegEx_GetMatchPosition( Index As Integer )
  Parameters 
Index
The match index to retrieve information.

  Returns

Returns the index where the match occurred.
  Exceptions

Code Exception Condition
209 MatchIndexOutOfBounds
 Index is greater than the number of matches found.
  Description

Returns the index in the string where the match was successful.

  Example Code 
Sentence$ = "In computing, a regular expression provides a concise and flexible means to " + Chr$(34) + "match" + Chr$(34) + " (specify and recognize) strings of text"
Pattern$ = "[A-Za-z]+" // Find one or more characters that are A-Z or a-z.

Print "Here are the following words contained in this sentence:"
Print Sentence$
Print

Rem Get the number of matches in the sentence.
MatchCount = SF RegEx_Match(Sentence$,Pattern$)

Print "The sentance contains " + Str$(SF RegEx_GetMatchCount()) + " words."
Print

Rem Iterate through the words listing them on the screen.
For X = 0 To MatchCount-1
    Print "Word: " + SF RegEx_GetMatchStr(X) + " | Position: " + Str$(SF RegEx_GetMatchPosition(X))
Next X

Wait Key
