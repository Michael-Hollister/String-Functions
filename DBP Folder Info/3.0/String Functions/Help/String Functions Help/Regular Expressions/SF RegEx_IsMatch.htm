SF RegEx_IsMatch

Returns a Boolean value indicating whether the pattern was found in the source string.

  Syntax 
Return Value As Boolean = SF RegEx_IsMatch( Str As String, Pattern As String )
  Parameters 
Str
Source string. 
Pattern
The regular expression.

  Returns

Returns 1 if the source string fully satisfied the pattern, otherwise it returns 0.
  Exceptions

Code Exception Condition
208 InvalidExpression
 Pattern is a invalid expression.
210 UnknownError
 Occurs if the complexity of matching the expression against an N character string begins to exceed O(N2), or if the program runs out of stack space while matching the expression, or if the matcher exhausts its permitted memory allocation. (this exception is unlikely to happen)
  Description

Returns a Boolean value indicating whether the pattern was found in the source string.

  Example Code 
Rem The following expressions were taken from regexlib.com

Rem Valid IP addresses.
IP1$ = "255.255.255.02"
IP2$ = "192.168.0.136"

Rem Invalid IP addresses.
IP3$ = "256.1.3.4"
IP4$ = "10.57.98.23."

Email1$ = "aa@bb.cc" // Valid email
Email2$ = "-a@b-.c-" // Invalid email

Rem Author: Andrew Polshaw 
IPPattern$ = "^(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])\.(\d{1,2}|1\d\d|2[0-4]\d|25[0-5])$"

Rem Author: arturo
EmailPattern$ = "^((([a-z0-9])+([\w.-]{1})?)+([^\W_]{1}))+@((([a-z0-9])+([\w-]{1})?)+([^\W_]{1}))+\.[a-z]{2,3}(\.[a-z]{2,4})?$"


Rem Validate the IP addresses.
Print "IP address validation."
Print "IP address " + IP1$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(IP1$,IPPattern$))
Print "IP address " + IP2$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(IP2$,IPPattern$))
Print "IP address " + IP3$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(IP3$,IPPattern$))
Print "IP address " + IP4$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(IP4$,IPPattern$))
Print

Rem Validate email addresses.
Print "Email address validation"
Print "Email address " + Email1$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(Email1$,EmailPattern$))
Print "Email address " + Email2$ + " : Valid = " + SF BoolToStr(SF RegEx_IsMatch(Email2$,EmailPattern$))
