SF RegEx_IsPartialMatch

Returns a Boolean value indicating whether the string partially satisfied the expression pattern.

  Syntax 
Return Count As Integer = SF RegEx_IsPartialMatch( Str As String, Pattern As String )
  Parameters 
Str
Source string. 
Pattern
The regular expression.

  Returns

Returns 1 if the source string partially satisfied the pattern, otherwise it returns 0.
  Exceptions

Code Exception Condition
208 InvalidExpression
 Pattern is a invalid expression.
210 UnknownError
 Occurs if the complexity of matching the expression against an N character string begins to exceed O(N2), or if the program runs out of stack space while matching the expression, or if the matcher exhausts its permitted memory allocation. (this exception is unlikely to happen)
  Description

Returns a Boolean value indicating whether the string partially satisfied the expression pattern.

  Example Code 
String$ = "0010111010100010010101000011101011101100101011110100101010"
Pattern$ = "1001|001"

Print "Match results"
Print "Full match: " + SF BoolToStr(SF RegEx_IsMatch(String$,Pattern$))
Print "Partial match: " + SF BoolToStr(SF RegEx_IsPartialMatch(String$,Pattern$))
