SF RegEx_Replace

Replaces the part of the string with the formatter where the match was successful.

  Syntax 
Return Str As String = SF RegEx_Replace( Str As String, Pattern As String, Formatter As String )
  Parameters 
Str
Source string. 
Pattern
The regular expression. 
Formatter
The replacement string.

  Returns

Returns the new string with the replacement strings where the match occurred.
  Exceptions

Code Exception Condition
208 InvalidExpression
 Pattern is a invalid expression.
210 UnknownError
 Occurs if the complexity of matching the expression against an N character string begins to exceed O(N2), or if the program runs out of stack space while matching the expression, or if the matcher exhausts its permitted memory allocation. (this exception is unlikely to happen)
  Description

Replaces the successful matches with the replacement string.

  Example Code 
String$ = "This is   text with   far  too   much        whitespace."
Pattern$ = "\s+";

Print "The old string: " + String$
Print
Print "The new string: " + SF RegEx_Replace(String$,Pattern$," ")
