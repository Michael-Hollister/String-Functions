SF Path_GetFileExtension

Get's the path's file extension.

  Syntax 
Return Path As String = SF Path_GetFileExtension( Path As String )
  Parameters 
Path
The path string to check.

  Returns

Returns the file extension.
  Exceptions

Code Exception Condition
205 InvalidPathCharacters
 Path contains one or more invalid characters. The invalid characters are listed here
206 MaxPath
 Path length is greater than 260. If the path is a folder, then it must be less than 248.
207 ReservedFileName
 Path contains one or more reserved file names. Reserved file names are listed here

  Description

Get's the path's file extension.

  Example Code 
String$ = "C:\Documents\Textfile.txt"
Ext$ = SF Path_GetFileExtension(String$)

Print "String evaluated = " + String$
Print "File extension is " + Ext$
Wait Key
