SF Path_Valid

Checks to see if the string is a invalid path.

  Syntax 
Return Path As String = SF Path_Valid( Path As String )
  Parameters 
Path
The path string to check.

  Returns

Returns 1 if the path is a valid path, otherwise it will return the following values.

  • 0 : Null Path
  • -1 : Invalid Characters
  • -2 : Max Path
  • -3 : Reserved Filename
  Description

Checks to see if the string contains invalid path characters or reserved filenames. This also makes sure the string is less than the max length of path strings (260).
  Example Code 
String$ = "C:\Documents\Textfile.txt"
String2$ = "C:\Documents\Information"
String3$ = "C:\PATH\<>Windows<>\**DoesNotWork**.L2T"
String4$ = "C:\ReservedName\NUl\image.bmp"

Valid = SF Path_Valid(String$)
Valid2 = SF Path_Valid(String2$)
Valid3 = SF Path_Valid(String3$)
Valid4 = SF Path_Valid(String4$)

Print "String evaluated = " + String$ + " : Path valid = " + Str$(Valid)
Print "String evaluated = " + String2$ + " : Path valid = " + Str$(Valid2)
Print "String evaluated = " + String3$ + " : Path valid = " + Str$(Valid3)
Print "String evaluated = " + String4$ + " : Path valid = " + Str$(Valid4)
