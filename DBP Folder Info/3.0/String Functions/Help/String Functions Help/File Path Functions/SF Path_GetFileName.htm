SF Path_GetFileName

Get's the path's file name.

  Syntax 
Return Path As String = SF Path_GetFileName( Path As String  )

Return Path As String = SF Path_GetFileName( Path As String, Mode As Boolean )
  Parameters 
Path
The path string to check. 
Mode
If true this will return the filename with the file extension.

  Returns

Returns the file name.
  Exceptions

Code Exception Condition
205 InvalidPathCharacters
 Path contains one or more invalid characters. The invalid characters are listed here
206 MaxPath
 Path length is greater than 260. If the path is a folder, then it must be less than 248.
207 ReservedFileName
 Path contains one or more reserved file names. Reserved file names are listed here

  Description

Get's the path's file or folder name.

  Example Code 
String$ = "C:\Documents\Textfile.txt"
Name$ = SF Path_GetFileName(String$)
Name_ext$ = SF Path_GetFileName(String$,1)

Print "String evaluated = " + String$
Print "File name is " + Name$
Print "File name with extension is " + Name_ext$
Wait Key
