SF Mixedcase

This will convert the string into mixed case representation of the string.

  Syntax 
Return Str As String = SF Mixedcase( Str As String, Mode As Boolean )
  Parameters 
Str
The string to convert. 
Mode
If the value is 0, this will convert the first letter of every word to a uppercase letter. If the value is 1, this will convert the first letter after every period. (".")

  Returns

Returns a mixed case string representation of the string.

  Description

This will convert the string into mixed case representation of the string. Mode 1 should be used when the string should be in a sentence format.

  Example Code 
Print SF MixedCase("the equality operators for dbpro are. { =,<>} THE INIQUALITY OPERATORS for DBPRO ARE { <, >, <=, >= }",1)
Print SF MixedCase("SET OBJECT POSITION",0)
