SF ToCharArray

Fills the array with each array index containing one character from the source string.

  Syntax 
SF ToCharArray( Arr As Array, Str As String )
  Parameters 
Arr
The array to fill. 
Str
Source string.

  Returns

This command does not return a value.
  Exceptions

Code Exception Condition
204 ArrayNotString
 Arr is not a string array.

  Description

Fills the array with each array index containing one character from the source string.

If the array size is larger than the length of the source string, then the rest of the array indexes are set to null strings. If the array size is less than the length of the source string, it will only hold the amount of characters that can be contained in the array size.

  Example Code 
String$ = "DarkBASIC"
Dim Array(Len(String$) - 1) As String

SF ToCharArray Array(),String$

Print "The string conatins the following characters."
For X = 0 To Array Count(Array())
    Printc Array(X) + " : "
Next X
