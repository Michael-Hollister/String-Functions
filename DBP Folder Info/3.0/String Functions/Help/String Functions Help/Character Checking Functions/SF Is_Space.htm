SF Is_Space

Checks to see if the character is white-space character.

  Syntax 
Return Value As Boolean = SF Is_Space( Str As String, Index As Integer )
  Parameters 
Str
The source string to check. 
Index
The position in the string to check.

  Returns

Returns 1 if the character is a white-space character. Returns 0 if the character is not a white-space character.
  Exceptions

Code Exception Condition
200 MustBePositiveValue
 Index is negitive.
201 IndexOutOfRange
 Index is greater than the length of the string.

  Description

Checks to see if the character is a white-space character.

The following white-space characters includes...

  • Space (0x20)
  • Horizontal Tab (0x09)
  • Newline (0x0A)
  • Vertical Tab (0x0B)
  • Feed (0x0C)
  • Carriage Return (0x0D)
  Example Code 
#Constant CRLF Chr$(0x0D) + Chr$(0x0A)
Rem Or you can use Crlf$() command from Matrix1 Utility Plug-ins.


String$ = "3..." + CRLF + "2..." + CRLF + "1... " + CRLF + "0!"
For X = 1 To Len(String$)
    If SF Is_Space(String$,X) = 1
        Inc Space_Count 
    Else
        Inc Non_Space_Count
    EndIf
Next X

Print "String evaluated = " + String$
