SF Is_Lower

Checks to see if the character is a lowercase alphabetic letter.

  Syntax 
Return Integer = SF Is_Lower( String, Integer )
  Parameters 
String
The source string to check. 
Integer
The position in the string to check.

  Returns

Returns 1 if the character is a lowercase alphabetic letter. Returns 0 if the character is not a lowercase alphabetic letter.
Returns -1 if the position is less than the length of the string. Returns -2 if the position is greater than the length of the string.

  Description

Checks to see if the character is a lowercase alphabetic letter.

  Example Code 
String$ = "DarkBASIC Professional"
For X = 1 To Len(String$)
    If SF Is_Lower(String$,X) = 1
        Inc Lower_Count 
    Else
        Inc Non_Lower_Count
    EndIf
Next X

Print "String evaluated = " + String$
Print
Print "This string has " + Str$(Lower_Count) + " lowercase characters."
Print "Non lowercase characters = " + Str$(Non_Lower_Count)

Wait Key
